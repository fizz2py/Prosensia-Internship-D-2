# Prosensia Internship - Day 2

This repository contains my Day 2 internship task for the Prosensia Internship Program. The project demonstrates the implementation of a **Linked List** along with the **Observer Design Pattern** in C++.

## Project Overview

The application implements a custom linked list data structure where observers can monitor changes made to the list. Whenever a node is added, removed, or modified, the registered observers are notified.

This project is built using C++ and CMake.

## Features

- Custom Singly Linked List
- Node insertion
- Node deletion
- Node traversal
- Observer Pattern implementation
- Automatic observer notifications
- Modular code structure using header and source files
